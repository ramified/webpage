This is my card game. It is still bad-behaved. See the ideal states and a failed try to make it draggable. It does not support any smartphones or tablets (since I'm lazy).

This card game allows you to generate different Auslander-Reiten quivers. To manipulate the quiver, simply click and drag the colored dots. Please note that the drag distance for vertical manipulation is relatively long compared to the horizontal drag distance.

Enjoy the game and have fun exploring different Auslander-Reiten quivers!
