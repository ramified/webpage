《The Equivariant Tubular Neighborhood Theorem》
This is the final project for manifold course. In this article, I introduce the necessary tools and prove the equivariant tubular neighborhood theorem from the simplest case to the most general case.
《Schur-Horn theorem》
This is the final project for Hua-seminar course. In this article, we state the Schur-Horn theorem, a amazing theorem in Linear Algebra. Instead of using the techniques in Linear Algebra, we introduce the Atiyah-Guillemin-Sternberg convexity theorem from the field of symplectic geometry, and after that we prove the Schur-Horn theorem as a corollary.
《Schur-Weyl duality》
In this final project for the geometry seminar course, We prove the Schur-Weyl duality by using the Double Centralizer Theorem. There are still some mistakes. By writing this document, I began to learn the invarient theory.
